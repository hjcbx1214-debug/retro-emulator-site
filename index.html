About the App

Retro Emulator is a mobile retro game emulator that lets you import local ROM files and play classic games on your phone. It supports multiple platforms including GBA and PSP, with features such as save states, on-screen controls, and gamepad mapping.

Key Features

  • Import game ROM files from local storage
  • Import game files from Google Drive (read-only; requires Google account sign-in)
  • Game library management and quick launch
  • Customizable on-screen controls and save slot management
  • Bluetooth and USB game controller support

Google Drive Permission

When you choose to import from Google Drive, the app requests read-only access to your Google Drive files (drive.readonly). This permission is used only to list and download game files you select to your device. The app does not modify or delete files in your cloud storage, and does not access files you have not explicitly chosen.
